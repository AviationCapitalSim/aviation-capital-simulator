## 1. Contexto General

Durante las pruebas de integridad del sistema de reset completo (ACS_MasterReset) se detectó un comportamiento aparentemente persistente en el Dashboard donde la base aérea:

Napoli (LIRN)

seguía apareciendo incluso después de ejecutar un reset total del simulador.

Inicialmente se sospechó de:

- Persistencia en localStorage
- Persistencia en IndexedDB
- Datos cacheados por Service Worker
- Persistencia en backend (Google Sheets)

Sin embargo, tras la auditoría del código del Dashboard se confirmó que el valor estaba **hardcodeado directamente en el HTML**.

---

## 2. Problema Detectado

### ❌ Hardcoded Base in Dashboard

El Dashboard incluía el siguiente bloque:

```html
<div class="welcome-chip base-chip">
<span id="chipBase">Base: Napoli (LIRN)</span>
</div>
