# ACS TECHNICAL SESSION LOG  
ID: TSL-2026-02-28-A  
Módulo: Buy New / Order Book Architecture / Finance Validation  
Estado: ABIERTO

---

## 1. Contexto General

Durante la revisión del módulo **Buy New Aircraft (acs_buy_aircraft.js v2.3)** se detectaron tres puntos críticos:

1. Error de sintaxis por desbalance de llaves en bloque BUY.
2. Falta de validación estructural de capital antes de registrar orden OEM.
3. Ausencia de visibilidad profesional de órdenes (Aircraft On Order).

Este TSL documenta las correcciones estructurales y la propuesta arquitectónica adoptada.

---

## 2. Problemas Detectados

### ❌ 2.1 SyntaxError en acs_buy_aircraft.js

Error en consola:

SyntaxError: Unexpected token ')'

Causa:
- Bloque `if (op === "BUY")` abierto sin cierre antes de iniciar bloque `if (op === "LEASE")`.

Impacto:
- El listener confirmBtn no se compilaba.
- Buy New quedaba inoperativo.

---

### ❌ 2.2 Falta de Validación de Capital (OEM)

El sistema permitía:
- Registrar órdenes.
- Crear entradas en `ACS_PendingAircraft`.

Aunque el capital fuera insuficiente.

Used Market ya tenía validación estructural.
Buy New no.

Riesgo:
- Capital negativo.
- Desalineación con Financial Core.

---

### ❌ 2.3 Ausencia de Order Book

Actualmente:

- Las órdenes se guardan en `ACS_PendingAircraft`.
- No existe módulo visual "Aircraft On Order".
- My Aircraft no diferencia correctamente:
  - Active
  - Delivered
  - On Order

Arquitectónicamente incompleto para un simulador Airbus/OCC-style.

---

## 3. Acciones Correctivas Implementadas

### ✅ 3.1 Corrección Estructural de Llaves

Se agregó cierre correcto del bloque:

} // cierre if (op === "BUY")

Antes del bloque:

if (op === "LEASE")

Resultado:
- Eliminado SyntaxError.
- Confirmación de orden funcional.

---

### ✅ 3.2 Implementación de Validación de Capital (OEM)

Se agregó:

```js
const finance = JSON.parse(localStorage.getItem("ACS_Finance") || "{}");
const currentCapital = Number(finance.capital || 0);

if (currentCapital < initialPay) {
  alert("❌ Not enough capital to place this order.");
  return;
}
